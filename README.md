# CarLane_Detect
期末大作业push到github  
一个简单的车道线检测程序

### 实现过程
* 图像灰度化、二值化
* 平滑图像处理（腐蚀、膨胀）
* ROI区域提取
* Canny边缘检测
* 霍夫线变换检测直线
* 最小二乘法拟合直线
* 视频按帧处理
